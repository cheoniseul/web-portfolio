# Web Portfolio

프론트엔드 개발자 천이슬의 개인 웹 포트폴리오입니다.  
About, Skills, Projects로 구성된 정적 웹 사이이며  
직접 디자인한 UI 요소와 애니메이션, 반응형 구조를 중심으로 구현했습니다.

---

## 기술 스택
- HTML  
- CSS  
- JavaScript  

---

## 주요 기능
- **반응형 레이아웃 구현**  
  다양한 화면 크기에서 무너짐 없이 자연스럽게 보여지는 구조 설계

 **히어로 섹션 ‘Portfolio’ 글자 애니메이션**  
  페이지 진입 시 텍스트가 등장하는 인터랙션 효과 적용

- **스크롤 등장 효과(Scroll Animation)**  
  스크롤 위치에 따라 섹션들이 부드럽게 나타나는 사용자 경험 제공

- **프로젝트 모달(Modal) 기능**  
  프로젝트 이미지를 클릭하면 슬라이드 이동이 가능한 상세 모달 팝업 제공

  ---

## 주요 섹션 구성
- **About** : 간단 프로필
- **Skills** : 사용 기술 소개
- **Projects** : 프로젝트 화면 및 요약 정보 제공
