# Web Portfolio

프론트엔드 개발자 천이슬의 개인 웹 포트폴리오입니다.  
About, Skills, Projects로 구성된 정적 웹 페이지이며  
직접 디자인하고 구현한 UI 요소 중심으로 구성되어 있습니다.

---

## 기술 스택
- HTML  
- CSS  
- JavaScript  

---

## 주요 구성
- **About** : 자기소개
- **Skills** : 사용 기술 소개
- **Projects** : 1차 · 2차 팀 프로젝트 요약 및 링크
